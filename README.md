# 艦これ遠征管理用タイマー

## 機能

- 一定時間経過後に通知するタイマー
  - 通知方法は現在は slack 経由を検討中
  - タイマーの取り消しも可能
- 時間は記憶させておくことが可能
  - 頻繁に使う物はワンクリックでタイマー開始が可能

## 構成要素

検討中の物含む

### フロントエンド

- React+TypeScript
- MUI
- React-Hook-Form

### バックエンド、インフラ

- Amplify 使って GraphQL とホスティング
  - データを参照し、通知を飛ばす箇所は EventBridge+Lambda
- 後で構成変えてサーバたてるかも
